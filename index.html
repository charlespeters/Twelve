The Grid

Really Simple SASS Mixins & Extends to Create a Grid Layout

First Off, Why?

Seriously, there are 9000 grid frameworks on Github, there is a herd of vicious octocats trying to prevent designers from uploading anymore. They have sharp teeth, Charles, you're playing a dangerous game here.

Ugh, I know. But let me explain, so this is a brain. This is what happens inside mine when I try and solve front-end problems:

"I need a grid. What are really good grid frameworks? Uhhhhh....... Foundation? I don't have time or energy to walk through all Foundations documentations. NEXT! Susy? I don't think I wanna use Compass because installing it takes a phone call to the Vatican to request super special holy water to pour on my Macbook and then a few more hours for the Macbook to dry. NEXT! Bourbon Neat? I love Bourbon Neat! Bourbon and Autoprefixer hate each other, so not a smart idea. NEXT! Bootstrap? Bootstrap is getting faster and smarter, but I totally don't need all of it for this one thing."

After the internal monolog, I think maybe a home spun solution with no depencies that I can just use in my existing SASS files might just be the ticket. All those frameworks are great but a lot of the time it feels like bringing a drone plane to a knife fight. I needed something smaller that I could decide through @extends when to declare and keep stuff really semantic.

Well, My Interest is Peaked

Okay well, let's see here. Using it shouldn't be to difficult, all you need to do is define @extend %col-x-x to your classes.

I have a for loop running to create %partials to generate classes like %column-3-12 & %grid-container to get a nice list of partials to extend into your project.

The Other Way

Start by declaring @include grid--container($container); on your outer container/page-wrap/whatever you want to call it. You can pass it a max-width of whatever you'd like.

and for the columns of your grid

@include column-width($span-of-columns, $context, $margin);

$span is a number of columns across and $context is total columns in your grid.

How it Works

You may write some SASS like this:

.page-wrap {
    max-width: 70rem;
    margin: 0 auto;
    section {
        width: 74%;
        margin-right: 2%;
        float: left;
    }
    aside {
        width: 26%;
        float: right;
    }
    &::after {
        content: "";
        clear: both;
        display: block;
    }
}

Using Twelve you can write it a simpler:

.page-wrap {
    @extend %grid-container;
    section {
        @extend %column-8-12;
    }
    aside {
        @extend %column-4-12;
    }
}
